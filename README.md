# 🏠 SIGEVU - Sistema Integral para la Gestión de Viviendas

Sistema web desarrollado durante mis pasantías profesionales para optimizar la gestión, consulta y administración de información relacionada con beneficiarios, viviendas y proyectos habitacionales.

## 📋 Descripción

SIGEVU es una plataforma web diseñada para centralizar la información de beneficiarios y proyectos de vivienda, permitiendo una gestión eficiente de los datos, consultas rápidas y generación de reportes para la toma de decisiones.

El sistema fue desarrollado como parte de mis pasantías en CORPOLARA, aplicando buenas prácticas de desarrollo web, diseño de bases de datos y administración de información.

## 🚀 Características

* Gestión de beneficiarios.
* Registro y actualización de viviendas.
* Administración de proyectos habitacionales.
* Gestión de ubicaciones geográficas.
* Consultas avanzadas de información.
* Importación masiva de datos.
* Panel administrativo.
* Sistema de autenticación de usuarios.
* Diseño responsive.

## ⚙️ Instalación

1. Clonar el repositorio

```bash
git clone https://github.com/Agabo32/Sistema_Central_de_Vivienda.git
```

2. Crear la base de datos MySQL.

3. Importar el archivo SQL.

4. Configurar las credenciales en:

```php
config/database.php
```

5. Ejecutar el proyecto desde XAMPP o Apache.
